# Collapsible Content

## Usage

```html
<collapsible-content open>

  <div>
    Lorem ipsum dolor sit amet, consectetuer adipiscing elit. Integer rutrum, orci vestibulum ullamcorper
    ultricies.
  </div>

</collapsible-content>
```
